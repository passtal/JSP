

요청 경로 배정
요청 메소드 지정

- GET	: doGEt
- POST	: doPost
- PUT	: doPut
- DELETE: doDelete

- 요청처리 : HttpSE

## 요청 경로 매핑
1. @어노테이션
2. web.xml 설정

### 어노테이션으로 요청 경로 매핑
@webServlet("/요청경로")

### web.xml 에 요청 경로 설정
- src/main/webapp/WEB-INF/web.xml
- <servlet>으로 서블릿 등록
- <servlet-mapping> 으로 요청 경로 매핑

  <!-- 서블릿 등록 -->
  <servlet>
    <servlet-name>SampleServlet<servlet-name>
    <servlet-class>web.SampleServlet<servlet-class>
  </servlet>
  
  <!-- 서브릿 요청경로 매핑 -->
  <servlet-mapping>
  	<servlet-name>SampleServlet</servlet-name>
  		<servlet-name>
  	<servlet-mapping>
