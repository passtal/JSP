# 요청 메소드

- GET		: doGet
  * 용도		: 조회
  * 데이터	: URL 파라미터로 전달
  * 본문(body): X
  * 특징 		: URL 길이제한, 민감한 정보 전달 금지
  * 적용		: 주소 표시줄에 입력, <form method="get"> AJAX (비동기 요정)
  
   ** -------------------------- **
  
- POST		: doPost
  * 용도		: 등록
  * 데이터	: 본문에 포함
  * 본문(body): O
  * 특징 		: 데이터를 노출시키지 않고 등록 요청
  * 적용		: 주소 표시줄에 입력, <form method="get"> AJAX (비동기 요정)
  
    ** -------------------------- **
  
- PUT		: doPut
  * 용도		: 수정
  * 데이터	: 본문에 포함
  * 본문(body): O
  * 특징 		: 데이터를 노출시키지 않고 수정 요청
  * 적용		: AJAX (비동기 요정)
  
    ** -------------------------- **

- DELETE	: doDelete
  * 용도		: 삭제
  * 데이터	: URL 파라미터로 전달
  * 본문(body): X // (일부 허용)
  * 특징 		: 삭제 요청 전 확인 절차 필요
  * 적용		: AJAX (비동기 요정)
