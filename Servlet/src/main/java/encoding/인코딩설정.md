# Servlet Encoding
- 요청 인코딩 & 응답 인코딩

- 요청 인코딩
  1. 	클라이언트로부터 받는 요청 데이터의 인코딩을 설정
  2. 	클라이언트 to 서버 (클라이언트 → 서버)
	
- 응답 인코딩
  1. 	서버에서 클라이언트로 보내는 응답 데이터의 인코딩을 설정
  2. 	서버 to 클라이언트 (서버 → 클라이언트)
	
** 요청인코딩이 선행되어야 응답인코딩이 가능 ** 


## web.xml 을 통한 전역 설정
- 하위의 모든 Servlet에 인코딩을 일괄 설정 : web.xml에 필터를 추가하기

** 방법 1) Spring Framework를 사용한다 (<filter></filter><filter-mapping></filter-mapping>) **

** 방법 2) 커스텀 필터를 사용한다 ( @WebFilter("/*") ) **

- 주의사항

  1. 	setCharacterEncoding()은 param을 호출하기 전에 설정해야 함
  2. 	setContentType()은 writer를 호출하기 전에 설정해야 함
  3. 	필터를 사용하면 ** 모든 요청 ** 에 대해 자동으로 인코딩이 설정됨
