# 페이지 이동

## servlet 에서 이동하기
### forward 방식으로 이동
RequestDispatcher dispatcher = request.getRequestDispatcher("/경로");
dispatcher.forward(request, response);
		
### redirect 방식으로 이동
String root = request.getContextPath();
response.sendRedirect(root + "/경로");
---
## jsp 에서 이동하기
### forward 방식으로 이동
<jsp:forward page="경로">
	<jsp:param name="파라미터명" value="값"/>
</jsp:forward>

### redirect 방식으로 이동
<c:set var="root" value="${ request.getContextPath() }" />
<c:redirect url="${ root }/경로" />
